## Laravel CRUD example

This project is a basic CRUD application built with Laravel. It allows performing **Create**, **Read**, **Update** and **Delete** operations.

## Requirements

Make sure you have the following installed before getting started

* PHP >= 8.1
* Composer
* Sqlite
* Laravel 11.x
